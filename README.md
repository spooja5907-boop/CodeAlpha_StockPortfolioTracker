# 📈 Stock Portfolio Tracker

## 📌 Description
This is a simple Stock Portfolio Tracker developed in Python as part of the CodeAlpha Python Programming Internship.

## ✨ Features
- Stores stock prices using a dictionary
- Accepts stock names and quantities from the user
- Calculates investment for each stock
- Calculates the total portfolio investment
- Allows multiple stock entries until the user types `DONE`

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run
1. Run the Python program.
2. Enter a stock name (e.g., AAPL, TSLA).
3. Enter the number of shares.
4. Type `DONE` to finish and view the total investment.
